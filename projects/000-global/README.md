# 000-Global: Enterprise Architecture

This directory contains enterprise-wide architecture artifacts that apply across all projects.

## Contents

- **policies/** - Organization-wide policies, standards, and guidelines
- **external/** - External reference documents (regulations, standards, best practices)

## Purpose

Global artifacts define the architectural context, principles, and constraints that individual projects must follow. These are typically created once and referenced by all project-specific artifacts.

## Typical Artifacts

- Architecture Principles (ARC-000-PRIN)
- Enterprise Standards
- Regulatory References
- Best Practice Guidelines
