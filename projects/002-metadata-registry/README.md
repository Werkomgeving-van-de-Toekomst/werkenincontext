# Metadata Registry Service

## Project Overview

**Project ID**: 002
**Project Name**: Metadata Registry Service
**Created**: 2026-04-19
**Status**: Active

## Description

Centralized metadata management service implementing the Dutch Government Metamodel GGHH (Gemeentelijk Gegevensmodel Haagse Hoek). Provides ArangoDB-based graph storage, REST/GraphQL APIs, validation engine, GitOps sync, and Dioxus admin UI.

## Relationship to IOU-Modern

This project is a component of IOU-Modern (Project 001), providing the metadata catalog for all government information assets.

## Technology Stack

- **Language**: Rust
- **Database**: ArangoDB (graph database)
- **Frontend**: Dioxus (WebAssembly)
- **API**: REST (v2), GraphQL
- **Storage**: GitOps-based YAML definitions
