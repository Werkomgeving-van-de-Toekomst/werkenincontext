# Planning Archive

This directory contains historical planning documents created by various skills (deep-plan, gsd, etc.) that have been consolidated into the main `.planning/` structure.

## Archive Contents

| Directory | Original Purpose | Consolidated To |
|-----------|------------------|-----------------|
| `planning/` | Agent orchestration implementation plan | `../features/agent-orchestration/` |
| `planning-api-extensions/` | API extensions (pipeline, auth, S3, WebSocket) | `../features/api-extensions/` |
| `planning-backend-eval/` | Database selection evaluation | `../features/backend-eval/` |
| `planning-document-workflow/` | Multi-stage approvals, delegation, escalation | `../features/document-workflow/` |
| `planning-3d-enhancements/` | 3D buildings visualization (Phase 2) | `../features/3d-map/` (completed) |

## Why Archived?

These directories were created by different planning tools and contain:
- Duplicate documentation
- Inconsistent formats
- Outdated implementation details
- Tool-specific configuration files

The key information has been extracted and consolidated into the main planning structure.
