## Technical Implementation Notes

**Single-Layer Architecture (PITFALL-03 Avoided):**
- Implementation uses `map.setPaintProperty('building-3d', 'fill-extrusion-height', value)` to switch views
- 2D mode: height = 0 (flat footprints)
- 3D mode: height = `['coalesce', ['get', 'height'], 10]` (actual heights)
- Popup handler remains on `building-3d` layer throughout
- Same layer = same data = no state desync possible

**Dutch Localization:**
- Button labels: "2D Weergave" / "3D Weergave"
- Matches existing UI language
