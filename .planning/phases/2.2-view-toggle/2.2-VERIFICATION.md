## Potential Issues and Mitigations

### Warning 1: State Synchronization on Mount

**Issue:** The `use_effect` in Task 2 component reads localStorage but doesn't update the Rust-side `view_mode` signal.

**Impact:** Button label may show incorrect mode on page load if localStorage has '2d'.

**Mitigation:** The sync_script in `use_effect` (lines 367-386) directly calls JS to set 2D mode if needed. The button label will correct itself on first click.

**Recommendation:** Consider adding a bidirectional sync, but not critical for MVP.

### Warning 2: FilterPanel3D Position Adjustment

**Issue:** Task 3 moves FilterPanel3D from `top: 10px` to `top: 60px` but LayerControl3D position is not documented.

**Impact:** Potential UI overlap if LayerControl3D is also at top-right.

**Mitigation:** LayerControl3D is typically positioned via CSS, not inline styles. Verify during execution.

**Recommendation:** Minor - can be adjusted during execution if needed.
